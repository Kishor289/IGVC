# *ARNAB DATTA*

# Setup
`source IGVC1/install/setup.bash `

# Dependencies:

- ROS 2 Humble on Ubuntu 22.04 Jammy
- geometry_msgs
- turtlesim
- rclpy (Python ROS 2 client library)

# Task 1 :Teleporting the _Turtle_
 
## To Run:
- run `ros2 run turtleop teleport` in terminal
- Enter values of x, y and theta

- run `ros2 run turtlesim turtlesim_node` in a separate terminal

# Task 2 : Moving the _Turtle_ in a Circular Trajectory

## To Run:
- run `ros2 run turtleop circle`  in terminal
- Enter values of radius and velocity


- run `ros2 run turtlesim turtlesim_node` in a separate terminal
-  Use `Ctrl+C` in first terminal to stop the circular motion
